# Lee V1-CA1 project

This repository contains analysis code for the V1-CA1 project. 

## Data
Simultaenous electrophysiological recording in the primary visual cortex (V1) and hippocampus CA1 of freely moving rats performing a spatial alternation task on the W-track. Typically four Livermore polymer probes are implanted in a single rat (bilateral V1 and CA1). The data consists of sleep and run epochs. During sleep epochs, the animal is placed in a sleep box. During run epochs, the animal performs spatial alternation on the W-track to get milk reward.
