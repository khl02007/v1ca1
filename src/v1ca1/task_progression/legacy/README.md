# Legacy Task Progression Scripts

Use this directory for outdated task progression scripts that should stay in the repo for reference or reproducibility.

Guidelines:

- Move scripts here when they are no longer part of the active workflow.
- Mark moved scripts as legacy with a short module docstring.
- Avoid new feature work in legacy scripts.
- Only make minimal fixes needed for reproducibility or inspection.
- Prefer active replacements in `src/v1ca1/task_progression/` when available.
